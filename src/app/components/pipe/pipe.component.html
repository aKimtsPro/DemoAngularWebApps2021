Les pipes

Les pipes permettent de transformer une valeur afin d'en obtenir une nouvelle

Il existeplusieurs raisons d'utiliser/de créer des pipes:

  • formater un élément de facon particulère
  • filtrer une liste
  • transformer un élément en un autre

pour l'utiliser il suffit de faire suivre notre valeur de '| nom_pipe' et la transformation sera appliqué

le pipe 'date' (DatePipe)

{{ date | date:'dd/MM/y' }}

le pipe 'currency' (CurrencyPipe)

{{prix | currency:'EUR':'symbol'}}

le pipe 'lowercase' et 'uppercase'

{{toLower | lowercase }} - {{toUpper | uppercase}}

le pipe 'number'

{{ nbr | number:'1.2-2' }}

le pipe 'json'

{{ toJson | json }}

pipe custom 'bouteille'

{{ bouteille | bouteille }}
